# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using transactional data.

## Tools Used
- Python (Pandas)
- MySQL
- Power BI

## Project Workflow
1. Data cleaning using Python
2. Feature engineering
3. SQL business analysis
4. Dashboard creation in Power BI

## Key Insights
- Male customers generated higher revenue.
- Loyal customers contribute the highest spending.
- Young Adult customers generated the highest revenue.

## Dashboard
Power BI dashboard shows:
- Total customers
- Average purchase amount
- Revenue by category
- Subscription distribution
